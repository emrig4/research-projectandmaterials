Installation

Unzip the file you download after purchase . The following files and folders will be in the unzipped file:

  • Edu.zip
  • Licensing
  • Documentation

The theme can be uploaded in two ways:

  • FTP upload: Using your FTP program, upload the non-zipped Edu folder into the /wp-content/themes/ folder on your server.
  • WordPress upload: Navigate to Appearance > Add New Themes > Upload. Go to browse, and select the zipped Edu folder. Hit Install Now and the theme will be uploaded and installed.

Once the theme is uploaded, you need to activate it. Go to Appearance > Themes and activate Edu theme.

If you need help installing Wordpress see the link here : http://codex.wordpress.org/Installing_WordPress

After the install finish, go to Edu theme setting page, then cick Save All Change to import default value to database.

Install Sample Data:

Edu has included wordpress XML demo file, you can see at /assets/sample-data/ . Following this step to import demo data to your site:

  1. Go to Tools > Import
  2. Select XML demo file from /assets/sample-data/
  3. Submit Upload file and import.
  4. Assign posts to an exitsting account.
  5. Check to Download and import file attachments
  6. Submit and wait when WP tools download attachments to your wordpress site

Megamenu Setup

If you use stMegaMenu plugin, when active it the first thing to do is go to stMenu setting under Appearance tab. At the bottom select which menu will apply for mega menu

Setup menu

Mega menu is follow by the 12 columns grid.

  1. Look at the Shortcodes menu items, it's the first level of menu, check "use as Mega Menu" if you want it to be Mega Menu display
  2. Select the width for mega menu display, it's follow by 12 columns grid ( look at the grid image above ). If you select wrap columns is 12 - it's 12 columns grid ( 1170px )
  3. Now go for child items, imagine your megamenu containt 4 columns as the image above. Each column need an menu item to wrap it. So the logic should be: 4 * 3 = 12 . Each menu columns need 3 columns wrap. ( NOTE: You can select what number as you want - see mega menu video tutorials)

Shortcodes for mega menu

If you want use shortcode in mega menu, follow these steps below:

  1. Check "Use as Title Menu", the menu item setting will display like the image above.
  2. Here you go, the caption field is a column title
  3. Paste your shortcode to Description field, and it should display
